# Midnight Cafe

Midnight Cafe adalah game simulasi cafe 2D bertema cozy yang dibuat menggunakan Python dan Pygame untuk memenuhi tugas UAS Pemrograman Berbasis Objek (PBO)

## Deskripsi Game

Pemain berperan sebagai barista yang melayani pelanggan di sebuah cafe kecil dengan suasana malam. Pemain bisa membuat makanan dan minuman, melayani customer, dan melakukan upgrade pada cafe

Beberapa customer punya cerita dan perkembangan hidup masing-masing yang akan berubah seiring berjalannya permainan

## Fitur Game

- Sistem melayani customer
- Pembuatan makanan dan minuman
- Sistem pembayaran
- Upgrade furniture cafe
- Sistem perpindahan scene
- Customer biasa dan customer tetap
- Story progression customer
- Suasana cozy dengan tema malam

## Cara Menjalankan Game

Install pygame terlebih dahulu:

```bash
pip install pygame
```

Lalu jalankan game:

```bash
python main.py
```
